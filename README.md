## 🎯 The Grind Philosophy

- 🟢 **Consistency > Intensity** — a small daily problem beats a weekend cram session.
- 🧩 **Patterns over memorizing** — two pointers, sliding window, prefix sums, XOR tricks.
- 🪲 **Bugs are tuition** — every segfault is a lesson paid for in advance.
- 📅 **Show up anyway** — tired brain solutions count double.
